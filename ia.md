# Spira IA

## MVP 화면 구조

```
Home
Direction
Program Detail (모달)
Resources
```

---

## Home

메인 실행 화면. 오늘의 업무 확인부터 Routine System 관리까지 한 화면에서 처리한다.

### 구성

**상단**
* Workspace 이름
* 활성 Program 목록 (탭 또는 필터)

**오늘의 업무 섹션**
* 오늘 날짜 기준 자동 생성된 Task 목록
* 완료 체크

**Routine System 섹션**
* Routine System 목록 (카드 또는 리스트)
* Routine System 추가 버튼

**Routine System 생성/편집 (Home 내 모달)**
* 시스템 이름
* 반복 요일 선택
* Task 목록 편집
* 저장 시 오늘의 업무에 즉시 반영

---

## Direction

사업의 방향성과 Program 관리

### 구성

* 비전
* Program 목록
* Program 추가 버튼 → Program Detail 모달

---

## Program Detail (모달)

Direction에서 Program 선택 시 열리는 모달

### 구성

* Program 이름
* 기간
* 목표
* 연결된 Routine System 목록

---

## Resources

수익/비용 입력

### 구성

* 수익 입력
* 비용 입력

---

## 제외 (Phase 2+)
