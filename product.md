# Spira

## 한 줄 설명

Spira는 1인 창업가를 위한 사업 운영 OS이다.

## 문제

기존 일정 앱은 일정만 관리한다.

기존 할 일 앱은 실행만 관리한다.

노션은 자유도가 높지만 반복 업무 시스템을 관리하기 어렵다.

프로젝트 관리 툴은 팀 협업 중심으로 설계되어 있다.

1인 창업가는 사업의 방향성, 반복 업무, 비용과 수익을 하나의 공간에서 관리하기 어렵다.

## 목표

사업의 방향성 관리

반복 업무 시스템 관리

비용 및 수익 관리

를 하나의 서비스에서 통합한다.

## 핵심 철학

목표는 생각하고,

시스템은 반복하고,

성장은 누적된다.

사용자는 목표를 설정하고,

그 목표를 달성하기 위한 반복 시스템을 만들고,

시스템이 매일의 업무를 생성하는 구조를 사용한다.

## 핵심 구조

→ Program

→ Task

## 대표 사용 사례

Workspace

* 아이바이마이

Programs

* 앱 개발
* 인스타그램 운영
* 매거진
* 커뮤니티

Goal

* 7월까지 팔로워 1,000명 증가

Routine System

* 월요일: 주제 선정
* 화요일: 촬영
* 수요일: 편집
* 목요일: 업로드
* 금요일: 분석

Task

* 오늘 콘텐츠 촬영

## MVP 목표

사용자가

1. Workspace 생성
2. Program 생성
3. Goal 생성
4. Routine System 생성

을 할 수 있어야 한다.

그리고 Routine System을 기반으로 오늘의 업무가 자동 생성되어야 한다.
